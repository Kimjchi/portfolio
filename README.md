# magnum-opus

Personal portfolio website for Jeremy Kim — freelance fullstack developer based in Paris.

## Stack

- **Framework**: [TanStack Start](https://tanstack.com/start) (React 19 + SSR)
- **Routing**: [TanStack Router](https://tanstack.com/router) (file-based)
- **Styling**: Tailwind CSS v4 + custom CSS
- **Animations**: GSAP (ScrollTrigger, SplitText, ScrambleTextPlugin)
- **CMS**: Strapi (headless, self-hosted)
- **UI**: shadcn/ui + Radix UI
- **i18n**: react-i18next (EN/FR)

## Getting Started

```bash
npm install
npm run dev
```

## Environment Variables

```env
STRAPI_URL=http://localhost:1337
STRAPI_API_TOKEN=your_api_token
```

## Build

```bash
npm run build
npm run start
```

## Linting & Formatting

```bash
npm run lint
npm run format
```
